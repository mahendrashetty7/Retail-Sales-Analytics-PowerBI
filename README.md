# 📊 Retail Sales Analytics Dashboard | Power BI

## 📌 Overview

Developed an interactive Retail Sales Analytics Dashboard using Microsoft Power BI to analyze sales performance, customer behavior, inventory, products, and store operations.

## 🚀 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Dashboard Pages

- Sales Overview
- Store Performance
- Product Analytics
- Inventory Dashboard
- Customer Analytics

## 💼 Key Features

- Interactive KPI Cards
- Dynamic Filters & Slicers
- Drill-down Analysis
- Data Modeling
- DAX Measures
- Power Query Transformations
